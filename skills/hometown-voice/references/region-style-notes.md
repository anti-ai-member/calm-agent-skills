# Region Style Notes

Use this file as a light checklist, not a phrasebook. Prefer user-provided examples over guesses. Avoid cultural performance. Local warmth should help the task feel easier, not become the task.

## Universal guardrails

- Prefer light adaptation first.
- Ask for examples if uncertain.
- Avoid exaggerated accent spelling.
- Avoid stereotypes and national character claims.
- Reduce local flavor when clarity matters.

## Chinese-speaking contexts

- Mainland China: Ask for province, city, or dialect if the user wants strong local style. Use clear Mandarin by default.
- Taiwan: Prefer natural, gentle, conversational Mandarin. Use Taiwan-specific wording only when confident or user-provided.
- Singapore: Ask whether the user wants English, Mandarin, Singlish, or Singapore-style Chinese. Use light local warmth unless stronger style is requested.
- Hong Kong or Macau: Ask whether the user prefers Cantonese-flavored Chinese, English, or standard written Chinese.

## English-speaking contexts

- United States: Usually direct, friendly, practical, and encouraging. Ask about region or professional tone if needed.
- United Kingdom: Usually measured, concise, and slightly understated. Avoid parody or exaggerated slang.
- Singapore English: Can be warmer and more pragmatic; use Singlish only when explicitly requested.

## Europe

- France: Prefer polished, respectful, context-aware phrasing. Avoid fake French phrases unless the user asks.
- Italy: Prefer warm, expressive, human phrasing while keeping the answer structured.
- Germany: Prefer clear structure, precision, directness, and well-labeled tradeoffs. Avoid reducing the user to bluntness.

## East Asia

- Japan: Prefer respectful, considerate, context-sensitive wording. For workplace scenarios, keep phrasing careful and non-imposing.
- Korea: Prefer respectful, relationship-aware wording. Be concise and avoid forced romanized Korean unless requested.

## Strong-style rule

Use strong localization only when all are true:

1. The user explicitly asks for it.
2. The agent has enough examples or confidence.
3. The task is casual enough that style will not reduce accuracy.
