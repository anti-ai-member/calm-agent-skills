# Tone Ladder

## Soft

Use for preserving warmth or when power distance is high:

```text
I can help with this. To avoid missing the current deadline, could we confirm whether A or B should come first?
```

## Neutral

Use for most workplace chats:

```text
I can take this on. Please confirm whether A or B is the priority for today.
```

## Firm

Use when boundaries have already been ignored:

```text
I can complete A today. B will need to move to tomorrow unless another priority is dropped.
```
